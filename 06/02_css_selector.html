02_css_selector(선택자)

css selectro(선택자)는 서식을 적용하기 위해 가장 먼저 설정해야 하는 방법이며, 선택자의 종류로는 50가지 정도가 있다.

  • *클래스 선택자
  • *아이디 선택자
  • *태그 선택자
  • 전체선택자
  • *자손, 후손선택자
  • 속성선택자
  • 동위선택자
  • 구조선택자(일반, 형태구조)
  • *반응, 링크 선택자
  • 전,후문자 선택자
  • 부정선택자
  • *클래스 결합, 태그와 클래스 결합
  • *그룹선택자

1. 클래스 선택자(.class명)

클래스 선택자는 사용자가 지정한 범위에 서식을 적용하기 위한 선택자로서 언제든 중복해서 서식을 적용하고자 할 때 사용한다. 클래스 선택자는 이름앞에 '.'기호를 넣어서 작성하는 것이 특징이고, 웹페이지 서식 작성시 많이 사용되는 선택자중의 하나이다.

2. 아이디 선택자(#id명)

사용자가 지정한 범위에서 서식을 적용하기 위한 선택자로서 한번 적용된 서식은 재사용이 불가능한 선택자이다. 중복되지 않는 서식을 작성하기 위한 선택자이며, 주로 레이아웃이나 자바스크립트, 제이쿼리에서 사용자 동적 기능을 구현하고자 할 때 사용한다.

3. 태그선택자(tag명)

html문서내에 있는 사용자가 지정한 태그요소에 서식을 한꺼번에 적용하고자 할 때 사용하기 위한 선택자이며, 사용자가 일일히 찾아서 적용하지 않아도 되기 때문에 편하지만 주의해야 한다.(원하지 않는 부분까지 서식이 적용될 수 있기 때문)
