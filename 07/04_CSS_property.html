04_CSS_property(속성) 종류 알아보기

css언어는 '선택자{속성:값;}'으로 작성된다. 여기서 속성은 사용자가 선택한 요소에 다양한 서식을 적용하기 위한 종류에 해당된다.

css속성의 종류

  • 글자관련 : font-family(글자모양), font-size(글자크기), color(글자색), text-decoration(밑줄없음, 밑줄, 윗줄), text-transform(영문 대, 소문자 전환), font-style(글자 기울게, 또는 보통으로 전환), font-weight(글자 굵게)
  • 문단관련 : text-align(정렬), text-indent(들여쓰기), line-height(행간)
  • 크기관련 : width(가로크기), heigth(세로크기)
  • 테두리관련 : border(테두리 선) 예)border:두께 선모양 선색상
  • 배경관련 : background-color, background-image, background-size, background-position......
  • 위치관련 : position, float, transform:translate(가로위치, 세로위치)
  • 레이아웃관련 : flex, grid, display
  • 애니메이션 관련 : animation, transition, @keyframes{내용}

1. 글자관련 속성 사용하기 p.180

font-familt : 글자모양을 지정하고자 할 때 사용하는 속성이며, 한글폰트를 지정하고자 할 때는 '굴림체'처럼 인용부호('')를 붙이며, 영문글꼴에서는 공백이 있는 폰트명도 인용부호('')를 붙인다.

font-size : 글자크기를 지정하는 속성으로서 px, %, em, rem단위가 있다,=. 기본값은 px단위를 많이 사용하며, 브라우저에서 표시되는 기본값은 16px임.

text-transform : 영문 소문자를 대문자로, 대문자를 소문자로, 단어의 첫글자만 대문자로 전환해주는 속성.

글자서식 적용시 단축속성을 사용할 수도 있다.
